# code-to-svg

**A CLI tool to visualize code's line length and indentation using SVG.**

```shell
code-to-svg my-code-file my-code-file.svg
```

## Installing

```
npm install -g code-to-svg
```


## Usage

```
Usage:
  code-svg-stream <input-file> [output-file] [-s]

Options:
  -s, --silent     No output
```
